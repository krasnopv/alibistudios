# Sanity Schemas

This directory contains Sanity schema definitions that can be used in a separate Sanity project.

## Available Schemas

### 1. SEO Metadata (`seo-schema.js`)
Complete SEO metadata management for dynamic page SEO.

**Files:**
- `seo-schema.js` - Schema definition (full and simple versions)
- `README-seo.md` - Detailed documentation for SEO schema
- `../docs/SANITY_SEO_USAGE.md` - Integration guide for Next.js

**Usage:** Copy to your Sanity project and import in your schema configuration.

### 2. Countries (`countries-schema.js`)
Country-specific content management for tax rebate information.

**Files:**
- `countries-schema.js` - Schema definition
- `README-countries.md` - Documentation

### 3. Directors (`directors-schema.js`)
Director profiles with works and awards.

### 4. Projects (`projects-schema.js`)
Project/portfolio item schema.

### 5. Team (`team-schema.js`)
Team member profiles with roles and social media.

## Integration

After copying schemas to your Sanity project:

1. Import the schema in your Sanity configuration
2. Deploy to your Sanity Studio
3. Use the provided utilities in this Next.js project to fetch data

See individual README files for detailed setup instructions.
